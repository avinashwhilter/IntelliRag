How to Run IntelliRag Locally
==========================

Quick start instructions to bring the Docker-based Ragflow stack up on your local machine.

Steps to Run
------------

**1.**

Clone the repository to your local machine:

git clone https://github.com/avinashwhilter/ragflow.git

Copy Copied ✓

**2.**

Change into the \`docker\` directory:

cd ragflow/docker

Copy Copied ✓

**3.**

Start services using Docker Compose:

docker compose -f docker-compose.yml up -d
